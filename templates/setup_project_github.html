Fork the project repo on GitHub.
1. Fork repo na GitHup
2. Clone fork ze swojego GitHub'a na lokalny dysk
$ git clone git@github.com:your_name_here/repo_name.git
3. Utwórz loklany branch
$ git checkout -b name-of-your-bugfix-or-feature
4. Teraz możesz dokonywać lokalnie zmian
5. Po wprowadzeniu i przetestowaniu zmian, wypchnij je na GitHub'a
$ git add .
$ git commit -m "Opis wprowadzonych zmian"
$ git push origin name-of-your-bugfix-or-feature
Submit a pull request through the GitHub website.
